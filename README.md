# Task-Earnings Bot v3

Telegram-based "Perform Task & Earn" bot integrated with Monetag ads.

## Features
- Earn coins by watching ads (10 ads = 1 task = 200 coins)
- Referral rewards
- Manual withdrawal system
- Admin dashboard
- Monetag postback integration
